# Algorithm And Data Structure (C)

This repository contains a collection of programming exercises and small projects developed in C, with a focus on fundamental concepts in algorithms and data structures.

The work presented here was completed as part of academic practice and is intended to reinforce theoretical knowledge through implementation.

---

## Overview

The repository includes solutions to a variety of problems covering key topics such as:

- Binary Trees (BST operations, heap conversion, tree balancing)
- Graph algorithms (e.g., graph coloring, analysis)
- Matrix processing and traversal
- Recursion and backtracking techniques
- Combinatorial and optimization problems
- String and numerical computations

Each project is provided as a separate `.zip` file and is designed to be self-contained.

---

## Implementation Details

All programs are written in standard C and rely only on common libraries such as:

- `stdio.h`
- `stdlib.h`

The implementations emphasize:
- clarity of logic
- modular design
- correctness of algorithmic approach

In some cases, priority is given to readability and learning outcomes rather than full optimization.

---

## Usage

To compile and run any project:

1. Extract the desired `.zip` file  
2. Navigate to the project directory  
3. Compile using a C compiler (e.g., GCC):

bash
gcc main.c -o program
./program
